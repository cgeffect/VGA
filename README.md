# VGA
VGA是一组用于播放透明度动画和实现透明度转换的工具集</br>
VGADemo 播放5中类型的透明度视频动画实例</br>
VGAPlayer.framework 实现透明度视频的解码与渲染</br>
VGAMac macOS端用于不同透明度视频的播放与转换工具</br>
[VGAMac转换工具说明文档](https://github.com/cgeffect/VGA/edit/master/README_MAC.md)
